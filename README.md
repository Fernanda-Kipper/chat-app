<h2 id="clone">Clone</h2>

<h4> Prerequisites</h4>

- Node v16.13.2
- npm v8.1.2
- Angular CLI v14.0.4

<h4>Starting</h4>

```
git clone https://github.com/Fernanda-Kipper/chat-app.git
cd chat-app
npm install
ng serve
$OPEN http://localhost:4200
```
